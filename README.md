# 3D 模型生成 Web 应用

一个基于 Next.js 和 React Three Fiber 的 3D 模型生成 Web 应用，支持从图像或文本描述生成交互式 3D 模型。

## 🚀 功能特性

### ✨ 核心功能

- **双模式输入**：支持图片上传和文本描述两种方式
- **实时 3D 预览**：上传后立即显示 3D 效果
- **进度可视化**：详细的生成进度跟踪
- **交互式 3D 查看器**：完整的 3D 模型查看和控制功能
- **现代化 UI**：响应式设计，适配各种设备

### 🎯 用户体验

- **直观操作**：清晰的上传 → 生成 → 预览流程
- **实时反馈**：进度条、状态提示、错误处理
- **丰富示例**：文本输入提供多种示例提示词
- **拖拽上传**：支持文件拖拽和预览

## 🛠 技术栈

- **Next.js 15** - 全栈 React 框架
- **React Three Fiber** - 3D 图形库
- **Three.js** - WebGL 3D 引擎
- **Zustand** - 轻量级状态管理
- **Tailwind CSS** - 样式框架
- **Lucide React** - 图标库

## 📱 界面预览

### 主界面

- 顶部导航栏包含应用标题和参数设置
- 标签切换：图片生成 / 文本生成
- 左侧输入区域，右侧 3D 预览

### 图片生成模式

- 拖拽或点击上传图片
- 支持 JPG、PNG、WEBP 格式
- 实时图片预览
- 文件大小和格式验证

### 文本生成模式

- 500 字符限制的文本输入
- 实时字符计数
- 示例提示词快速选择
- Ctrl+Enter 快捷键提交

### 3D 查看器

- WebGL 渲染的 3D 场景
- 鼠标拖拽旋转视角
- 滚轮缩放
- 网格显示开关
- 自动旋转功能

## 🚀 快速开始

### 安装依赖

```bash
cd frontend
pnpm install
```

### 启动开发服务器

```bash
pnpm dev
```

### 访问应用

打开浏览器访问：http://localhost:3000

## 🎯 使用指南

### 图片生成 3D 模型

1. 选择"图片生成"标签
2. 拖拽或点击上传图片（支持 JPG、PNG、WEBP）
3. 等待图片上传和预览
4. 点击"开始生成 3D 模型"
5. 查看生成进度和 3D 结果

### 文本生成 3D 模型

1. 选择"文本生成"标签
2. 在文本框中输入描述（例如："一个可爱的机器人，银色金属外壳"）
3. 可以使用提供的示例提示词
4. 点击"生成 3D 模型"或按 Ctrl+Enter
5. 查看生成进度和 3D 结果

### 3D 模型交互

- **旋转**：鼠标左键拖拽
- **缩放**：鼠标滚轮
- **平移**：鼠标右键拖拽
- **网格**：点击网格按钮开关
- **自动旋转**：点击自动旋转按钮

## 🔧 开发说明

### 状态管理

使用 Zustand 管理全局状态，包括：

- 生成任务状态
- 输入数据（文件、文本）
- UI 状态（加载、预览等）

### 3D 渲染

- 基于 React Three Fiber 构建
- 使用 Three.js 进行 3D 渲染
- 支持 GLTF/GLB 模型格式
- 包含光照、材质、动画等效果

### 响应式设计

- 移动端优先设计
- 自适应布局
- 触摸手势支持
